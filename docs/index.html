Welcome to Sabre's Lair! Δ

</About>

Sabre is a chatbot written almost entirely in Node Javascript using the Discord.js Wrapper Library. With the exception of a few functions such as a built-in SQLite system, Sabre nearly runs purely over the Discord.js wrapper. Sabre is almost entirely maintained over Github through Atom.io.

A secure shell is used to access the server where the latest version of the program is loaded.

I would like to thank An Idiot's Guide for their examples in using the Discord.js wrapper, as well as my hand-picked team of developers helping me with tasks ranging from ideas to programming.

</PermissionSystem>

• (PL4O) Owner Only
• (PL4) Developers
• (PL3) Admin / Super Administrator
• (PL2) Moderator / Shadow Moderator
• (PL1) Shadowling / Citizen of Alaska
• (PL0) Nothing

The permission tier system (or Permission Level system) ensures that only users with the above specified roles have access to specific Sabre commands.

Sabre was originally intended to be used as a small-scale moderation system for functions relating to Cyber Security. It was then brought online for the small gaming community of Roblox Anchorage Alaska - soon after, Sabre exploded in popularity featuring many minigames. It was brought online August 4, 2017.
